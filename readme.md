# ing
Ping, but shorter output.

Installable with:
- [Zplug](https://github.com/zplug/zplug): `zplug "rummik/zsh-ing"`
- [Antigen](https://github.com/zsh-users/antigen): `antigen bundle rummik/zsh-ing`

Example:
```
> ing google.com
google.com: 13.7 ms
google.com: 13.3 ms
google.com: 13.3 ms
google.com: 13.6 ms
google.com: 14.9 ms
google.com: 13.8 ms
```
